【Stata】基準值設定2 (區間的平均值)

By YH Chen 31 October 2024
使用情境

之前在 【Stata】基準值設定 的文章中介紹過基準值的設定，但那時候只考慮了提取資料中某一個年份的值作為基準值

但考慮到有時候使用一段區間的平均值，來設定基準值可能會更符合資料分析應用的實際情況，因此在這篇文章是要示範如何計算資料中某一段區間的平均值，並將計算的結果作為基準值

原始資料

資料的使用，同樣沿用 【Stata】基準值設定 文章中的資料， 使用 世界銀行 (World Bank)的全球GDP資料

下載的資料經過簡單的整理後，只留下5個國家 2019-2022年的資料 (如下圖)

*圖片來源：YH Chen

Stata 語法

完整的 code 如下，可以分成兩個部分進行

1. 第一部分，計算2020-2021年每個國家的平均值

gen inrange = 0 

replace inrange = 1 if year >= 2020 & year <= 2021

bysort country: egen base_GDP = mean(GDP) if inrange==1

計算後的結果，可以看到 2019 年和 2022 年的為空值 (如下圖)，這時候如果直接開始進行資料分析，可能會造成分析結果有錯誤的現象，因此需要進一步的處理，也就是補資料的部分

*圖片來源：YH Chen

2. 第二部分，補資料

egen id_country = group(country)

tab country

local num = r(r)

display `num'

forv i = 1/`num' { 
    summarize base_GDP if inrange==1 & id_country==`i'
    local avg_GDP = r(mean)
    replace base_GDP = `avg_GDP' if id_country==`i'
}

補完資料後的結果 (如下圖)，可以看到原先 2019 年和 2022 年為空值的地方，也都補上了第一部分計算好的平均值，這時候再開始進行數據分析，就不會出現缺失值或者分析結果有錯誤的情形

*圖片來源：YH Chen

Stata 語法說明

第一部分的註解:

  1. 建立區間的標記 inrange

  2. 以 country 區分，使用 mean()，計算當 inrange==1 時，所有 GDP 的平均值

gen inrange = 0 

replace inrange = 1 if year >= 2020 & year <= 2021

bysort country: egen base_GDP = mean(GDP) if inrange==1

第二部分的註解:

  1. 使用 group()，為 country 進行編號

  2. 提取 country 個數，並建立成 local 變數 `num'

  3. 使用迴圈的方式將資料補上，這個地方的關鍵是在 summarize 這條 code，我們提取的是 inrange==1 的資料 (也就是第一部分計算好的平均值)，接著再建立 local 變數 `avg_GDP’，最後用 replace 搭配 if 的方式，將基準值的資料補進去原本為空值的資料格內


egen id_country = group(country)

tab country

local num = r(r)

display `num'

forv i = 1/`num' { 
    summarize base_GDP if inrange==1 & id_country==`i'
    local avg_GDP = r(mean)
    replace base_GDP = `avg_GDP' if id_country==`i'
}
